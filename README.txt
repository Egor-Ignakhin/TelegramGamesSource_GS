Редактирование games.json:
name - Название игры
desc - Описание игры 
playLink - Ссылка на игру (если игры еще нет - пустые ковычки)
imagePath - Путь до иконки игры    
buttonType - play-button(если игра вышла)/soon-button(если игры еще нет)
gameStatus - Играть(если игра вышла)/Скоро(если игры еще нет)
